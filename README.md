# African Climate Trend Analysis - Week 0

## Project Overview
This repository contains the analysis of historical climate and weather data for Ethiopia (Jan 2015 – March 2026). The project aims to identify long-term temperature and rainfall trends using NASA POWER data to support policy-driven climate resilience strategies.

## Environment Setup and Reproduction
To ensure a consistent development environment, this project uses a standardized Python setup and GitHub Actions for Continuous Integration (CI).

### Prerequisites
- Python 3.10 or higher
- Git
